Frequency

- Pin: 2, 4, 5, 6, 7, 10, 11, 12, 13, 14, 15, 18, 19, 22, 28, 34, 36, 38.

- Frequency: int positive

Amplitude

- i2c: 0, 1

- Channel: 0, 1, 2, 3

- Amplitude: int positive

Duration

- Duration: int positive
